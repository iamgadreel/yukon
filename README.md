# lecture
A simple lecture scheduling application

#Scope
Develop an API to publish lecture schedule of a university. API should display schedules based on time, batch and program. 

e.g. schedule for {module}, {batch} at {start time} to {end time} *In a json manner

** Test HTML page is provided
** DB dump is provided

Completed with 
  -Tomcat 7.0.77
  -MySQL
  -Spring Tool Suit 3.8.4
